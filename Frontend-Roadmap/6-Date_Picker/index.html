About This Component

  • Static UI: This is a non-functional datepicker built with HTML & CSS only
  • Absolute Positioning: The calendar dropdown uses absolute positioning
  • CSS Grid: Days are laid out using CSS Grid (7 columns)
  • Responsive Design: Adapts to different screen sizes
  • Future Enhancement: Can be made functional with JavaScript
